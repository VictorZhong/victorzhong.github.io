暴雨在左，暴晒在右——黑排角海岸线穿越

· 2022-08-20 · # 广东 # 徒步 # 海岸线


总体而言，这条路线是一条老少咸宜且风景优美的入门级路线，全程大概8km，慢一点的脚程三个钟绰绰有余，且全程贴着海岸线，潮湿而微腥的海风，伴随着海浪拍打黑礁石的声音，十分惬意。

路线

  1. 🚙广州 -> 惠东。 ~3.5h
    直接导航到 “悦海明珠”饭店，这是个挨着沙滩的饭店，穿过餐厅就能出去盐洲岛海滩，离徒步起点也近，点几个海鲜、农家小炒，吃饱喝足，一会路上带两支水轻装上阵即可。
  2. 🚶盐洲岛海滩 -> 湾仔沙滩 -> 盐洲岛彩石滩 -> 黑排角 -> 杨屋沙滩 -> 黑排角帐篷营地 -> 双月湾滑翔伞基地。 ~3h

    从饭店出发，走一小段水泥路就能到湾仔沙滩，这里是徒步的起点，简单热身，然后就一路沿着海岸线走就完事了。这是条很成熟的路线，很多户外俱乐部都在大大小小的岔路口做了标记，看着绑着小彩带的树枝走就没错了。

  • 彩石滩（非常适合拍照）


    非常专业的领队😜

  • 黑排角。据说黑排角名字的由来，就是因为这里一排黑色的礁石，与深圳的“红排角”相呼应。

  • 山路其实比较不好走，尤其是中间两次大暴雨突袭，没有雨衣的朋友全成了落汤鸡，衣服干了湿湿了干😂


在抄一段近路的时候，一个猛子踩到灌木丛里结果一脚踩空，挣扎着出来后发现不太对劲，眼镜没了，凉凉🤣🤣
没想到Cloudy姐就在后面，她果断发动了前后几个壮丁一起帮忙找，我都基本不抱希望了，因为这丛生的杂草都半人高了，加之这会快傍晚了光线也不好——结果，地毯式搜索了十分钟竟然真的找到了！！

p.s. Cloudy姐是因为一次性雨衣认识的，她当时在问哪里有卖一次性雨衣，而我刚好有多了一件...😏

  • 中间甚至有个便利店，所以真的不用带补给。

  • 双月湾 是徒步终点，也是个沙滩，不过开发得早，人明显多，旁边是个滑翔伞基地，八九百滑一次，全程四十多分钟，但真正滑翔只有十分钟不到吧。

  1. 车子停在东和小学门口，所以最后又走了 40min 左右从双月湾到停车场，最后稍事休整，回返广州。一路秋风和煦，晚霞宜人。
